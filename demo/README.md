Prototype structure

- `index.html`: standalone premium landing concept (hero, features, journey, proof section).
- `styles.css`: full visual system (light editorial palette, typography, responsive layout, motion states).
- `app.js`: reveal animation, count-up metric, and GSAP ScrollTrigger integration.

Usage

1. Open `demo/index.html` in browser.
2. Existing `src` code remains untouched. This folder is isolated for design validation.

Upgrades included

- Typography upgrade: `Newsreader` + `Manrope`.
- Motion upgrade: GSAP + ScrollTrigger via CDN.
- Visual realism: photographic hero/proof media and reduced template-like gradients.
